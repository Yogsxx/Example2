List Element adalah untuk membuat daftar list. Disini dibagi menjadi 3 macam.
1. Ordered List, yaitu list yang membuat daftar yang berurutan.
2. Unordered List, yaitu list yang tidak berurutan.
3. Description List, yaitu list untuk membuat deskripsi sebuah kata.

Contoh:
1. Ordered List

    Hal-hal yang perlu dipalajari dalam membuat web development.
  1. HTML
  2. CSS
  3. Java Script
  4. PHP
  5. MysqL

2. Unordered List
    Nama aplikasi internet yang ada di Dunia
  • Google Chrome
  • Mozila FireFox
  • Microsoft Edge

3. Description List
HTML
HTML adalah Hyper Text Mark Up Language.
